# ActiveAI

ActiveAI is a production-ready, gamified productivity and personal growth platform built with **React + TypeScript + Vite + Tailwind CSS**.

## Features

- Dashboard with todayâ€™s tasks, XP progress, streak meters, and motivational insights.
- Task manager with priorities, due dates, completion tracking, and XP rewards.
- Habit tracker with streak counters and a 28-day calendar heatmap.
- Goals with milestones, progress %, completion state, and badge rewards.
- Pomodoro focus timer (25/5) with focus session logging and XP.
- Gamification system: XP, level ups, achievements, XP toasts, and confetti unlocks.
- Rule-based AI Coach panel with daily feedback, streak compliments, and improvement suggestions.
- Dark, modern UI using rounded cards, gradients, and smooth transitions.
- LocalStorage persistence for all app data.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
